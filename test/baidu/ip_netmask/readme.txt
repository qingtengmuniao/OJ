Problem Description
当今世界，网络已经无处不在了，小度熊由于犯了错误，当上了度度公司的网络管理员，他手上有大量的 IP列表，小度熊想知道在某个固定的子网掩码下，有多少个网络地址。网络地址等于子网掩码与 IP 地址按位进行与运算后的结果，例如：
子网掩码：A.B.C.D
IP 地址：a.b.c.d
网络地址：(A&a).(B&b).(C&c).(D&d)
Input
第一行包含一个整数，代表测试数据的组数，
接下来组测试数据。每组测试数据包含若干行，
第一行两个正整数。接下来行，每行一个字符串，代表一个 IP 地址，
再接下来行，每行一个字符串代表子网掩码。IP 地址和子网掩码均采用 的形式，其中均为非负整数，且小于等于255。
Output
对于每组测试数据，输出两行：
第一行输出: "Case #i:" 。代表第组测试数据。
第二行输出测试数据的结果，对于每组数据中的每一个子网掩码，输出在此子网掩码下的网络地址的数量。
Sample Input
2
5 2
192.168.1.0
192.168.1.101
192.168.2.5
192.168.2.7
202.14.27.235
255.255.255.0
255.255.0.0
4 2
127.127.0.1
10.134.52.0
127.0.10.1
10.134.0.2
235.235.0.0
1.57.16.0

Sample Output
Case #1:
3
2
Case #2:
3
4
