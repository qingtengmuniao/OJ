方法的确挺简单，按区间dp。首先将区间排序，按end时间点排，如果相等，则按start排，都是从小到大。
dp状态：考虑第i个区间后能获得最大权重。
dp初始化：初始化为第i个区间的权重。
dp转移方程：
if (interval[i].start - interval[j].end >= r) {
	dp[i] = max(dp[i], dp[j]+interval[i].weight);
}  
两层for循环，从头开始往后考虑，状态转移的时候，遍历此状态之前所有已经算好的值。
